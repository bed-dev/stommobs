# Examples

This directory contains copy-pasteable usage examples for `stommobs`.

- `examples/java/JavaBuildersExample.java` - Java Builders for profiles and spawning
- `examples/kotlin/KotlinDslExample.kt` - Kotlin DSL profile + spawning flow
- `examples/kotlin/VanillaQuickSpawnExample.kt` - minimal built-in profile spawning

These examples are written as library integration snippets and are not wired as a separate Gradle module.
Use them inside your Minestom project entrypoints/listeners.
